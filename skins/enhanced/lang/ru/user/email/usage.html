LANG_SENDER_NOTE=Отправитель: Значение 'От' устанавливается в заголовке. Это значение не должно рассматриваться окончательно, так как отправитель может указать любоей значение, какое захочет. LANG_AUTH_NOTE=Аутентификация: Если используется smtp аутентификация, это покажет использованное имя пользователя. Скрипты покажут владельца скрипта. Указывайте аккуратно. LANG_SENDER_HOST_NOTE=Host отправителя: Если это установлено, будет показан IP который подключался к серверу. Если IP не установлено, тогда email был отправлен через локальный скрипт. LANG_PATH_NOTE=Путь: Если email генерируется скриптом, значение пути покажет папку в которой выполнялся скрипт. Еmail в очереди будет содержать значение пути для каждой попытки повтора.
